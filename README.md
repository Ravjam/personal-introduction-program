# Personal Introduction Program

## Project Overview
This is a simple Python program that asks the user a few questions such as name, age, city, hobby, favorite food, and dream job.  
After collecting the answers, the program prints a friendly welcome message using the information provided by the user.

The purpose of this project is to practice basic Python concepts such as:
- input()
- variables
- print()

---

## Setup Instructions

1. Install Python on your computer.
2. Download or clone this repository.
3. Open the project folder.
4. Run the program using the command below.

python personal_intro.py

Note: The program was also tested using Google Colab.

## Technical Details

The program works in the following steps:

1. The program asks the user questions using input().
2. The answers are stored in variables.
3. The program prints a friendly message using print().

Variables used:
- name
- age
- city
- hobby
